
🚀 Crypto Smart Contract Project
-----------------------------------

📌 Overview
--------------
This project is a blockchain-based smart contract built using Solidity. It simulates a simple crypto/reward system where users can mint, transfer, and manage tokens securely.
The goal of this project is to understand how decentralized applications (dApps) work and to apply best practices in smart contract development, security, and gas optimization.

⚙️ Features
---------------

💰 Token Minting


Admin can mint new tokens to specific addresses




🔄 Token Transfer


Users can transfer tokens between accounts




🛑 Pause / Resume Contract


Admin can pause and resume contract operations




📢 Events Logging


Important actions emit events for tracking (minting, transfers, pause/resume)




🔐 Access Control


Only authorized users (admin) can perform sensitive operations




⚠️ Error Handling


Uses require, revert, assert, and custom errors for validation and safety
